# /lab-report — 实验报告全流程自动化技能

Claude Code 自定义技能，将大学实验报告从「读要求」到「生成 LaTeX」整合为一条命令。

## 能做什么

```
读实验要求 PDF → 输出实验概述 → 生成 TODO 列表 → 写全部代码 → 等用户编译跑完 → 核对数据 → 生成完整 LaTeX 报告
```

适用于所有「编写代码 → 运行测试/仿真 → 采集数据 → 撰写报告」的理工科实验报告场景。

## 安装

将本仓库克隆到 Claude Code 的 skills 目录：

```powershell
git clone https://github.com/fcsguyxcw/lab-report.git $env:USERPROFILE\.claude\skills\lab-report
```

安装后重启 Claude Code 或重新加载会话即可生效。

## 使用方式

### 全流程模式

在实验目录下直接运行：

```
/lab-report
```

技能会自动发现目录下的 PDF（实验要求）和 `.tex`（报告模板），然后逐步推进。

### 中断后继续

如果中途退出，已生成的文件（`src/`、`results/`）不会丢失，回来直接：

```
/lab-report 继续
```

技能会扫描已有文件，自动定位到下一个未完成的步骤。

### 只写代码

跳过实验概述，直接分析要求并生成所有代码到 `src/`：

```
/lab-report 只写代码
```

### 只写报告

代码和结果都已准备好，直接从 `results/` 生成 LaTeX 报告：

```
/lab-report 只写报告
```

## 使用场景

### 场景 1：从零开始的完整实验

```
实验/
  实验要求/           ← 放入实验指导书 PDF
  实验报告模板/        ← 放入学校模板 .tex 和图片资源
```

运行 `/lab-report`：
1. Claude 读取 PDF，输出实验概述（目的、环境、要求）
2. 生成 TODO 列表 + 所有代码模块到 `src/`
3. 告诉你去编译、运行、截哪些图、存到哪个文件名
4. 你完成后执行 `/lab-report 继续`
5. Claude 核对 `results/` 里的内容是否覆盖了模板所有章节
6. 生成完整 LaTeX 报告

### 场景 2：多周实验（按周推进）

```
实验/
  week1/
    实验要求/
    实验报告模板/
    src/
    results/
  week2/
    ...
```

在 `week1/` 下运行 `/lab-report`，完成第一周后进入 `week2/` 再次运行。每周独立。

### 场景 3：代码已经写完，只要报告

```
实验/
  src/                ← 已有代码
  results/            ← 已有截图和数据
  实验要求/
  实验报告模板/
```

直接运行 `/lab-report 只写报告`，跳过代码生成，从数据校验开始。

### 场景 4：只想看看要求和 TODO

运行 `/lab-report`，拿到 TODO 列表后，你可以不执行，先评估工作量。

## 目录约定（灵活）

优先自动发现，回退结构：

```
<实验目录>/
  实验要求/           ← PDF：实验指导书
  实验报告模板/        ← .tex 文件 + 图片资源（如校徽）
  src/               ← Claude 自动生成代码于此
  results/           ← 用户的截图、性能数据放这里
```

## 流程详解

| 步骤 | 做什么 | 谁做 |
|------|--------|------|
| 1. 自动发现 | 扫描 PDF 和 .tex | 🤖 |
| 2. 实验概述 | 实验目的 / 环境 / 要求（≤300字） | 🤖 |
| 3. TODO + 代码 | 列出分步任务，一次性生成全部代码到 `src/` | 🤖 |
| 4. 用户执行 | 编译、仿真、截图，结果存入 `results/` | 👤 |
| 5. 数据校验 | 对照模板章节检查数据是否齐全 | 🤖 |
| 6. 生成报告 | 按模板结构填充 LaTeX 报告 | 🤖 |

## 报告风格

- 保留模板的 `\usepackage`、字体设置、页眉页脚不动
- 代码片段用 `\begin{lstlisting}` 嵌入
- 截图自动引用 `\includegraphics{../results/xxx.png}`
- 性能数据用 LaTeX `\begin{table}` 制表
- 摘要自动写草稿，总结留白给用户补充

## 依赖

- Python 3 + pypdf（用于读取实验要求 PDF）
- LaTeX 环境（xelatex，用于编译报告）
- Claude Code
